# GrowthGuard
This project is about IoT plant monitoring system with a dashboard to show current information and statistics.

## Devices used
1. ESP32
2. DHT11 Sensor
3. Soil Moisture Sensor
4. Jumper Cable (Male and Female)
5. Breadboard
6. LED
7. Resistor 220 ohm
